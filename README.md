# Propspace-test

This is an aplication to manage internal information management system for a company.

  - Add Company Profile
  - Add the exactly point where your office is
  - adn you can access the widget.

### Requirements:
  - React >= 16.6.0
  - NPM >= 8.x
  - yarn >= 1.6.0

### Run Dev
```sh
$ cd prospace-test
$ yarn install
$ yarn start
```
### Run Test
```sh
$ cd prospace-test
$ yarn install
$ yarn test
```

### Build
```sh
$ cd dillinger
$ yarn build
```
